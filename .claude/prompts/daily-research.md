# Daily Technical Research Prompt

以下のトピックについて、最新の情報を Web 検索で調査し、日本語で要約してください。

## 調査トピック

### 1. Claude Code / Anthropic
- Claude Code の新機能・アップデート・リリースノート
- Claude API の変更点
- Anthropic 公式ブログの新着記事

### 2. エディタ技術
- Tiptap / ProseMirror の最新リリース・Breaking Changes
- CodeMirror の重要な更新

### 3. フロントエンド
- Next.js の新バージョン・重要な変更
- React の新機能・RFC
- MUI (Material-UI) の更新

### 4. VS Code 拡張開発
- VS Code API の変更・新機能
- Extension Host の更新
- Webview API の変更

### 5. モバイル
- Capacitor の新バージョン・変更点

## 出力フォーマット

各トピックについて以下の形式で出力してください:

- **更新があった項目のみ** 記載する（変更なしのトピックは省略）
- 各項目にソース URL を含める
- 影響度（高/中/低）を判定し、当プロジェクト（Tiptap ベースの Markdown エディタ）への影響があれば明記する

```markdown
## [トピック名]

### [項目名]
- **概要:** 1-2文の要約
- **影響度:** 高/中/低
- **当プロジェクトへの影響:** あれば記載
- **ソース:** URL
```
