# Business Thinking Methods Overview

Updated: 2026-03-21


## Overview

A categorized list of major thinking methods found in business books.\
Organized into 5 categories by purpose, referencing sources such as "Thinking Methods Encyclopedia" (60 patterns, Shoeisha).


## 1. Foundational Thinking (Building Blocks)

| Method | Description |
| --- | --- |
| Logical Thinking | Structured, contradiction-free reasoning. MECE and logic trees are core frameworks |
| Critical Thinking | Question assumptions and identify the essence. Verify with "So what?", "Why so?", "True?" |
| Deductive Thinking | Derive specific conclusions from general principles (syllogism) |
| Inductive Thinking | Derive general principles from specific facts |
| Abductive Thinking | Infer the most plausible hypothesis from observed results |
| Meta Thinking | Observe things objectively from a higher-level perspective |
| Abstraction Thinking | Abstract from specifics and apply to different contexts through round-trip reasoning |


## 2. Creative & Ideation Thinking

| Method | Description |
| --- | --- |
| Lateral Thinking | Generate ideas freely without being constrained by assumptions |
| Design Thinking | Start from user empathy, define problems, and validate with prototypes |
| Art Thinking | Create value from scratch, driven by inner impulse and aesthetic sense |
| Analogy Thinking | "Borrow and combine" knowledge from different domains to generate new ideas |
| Paradoxical Thinking | Consider the opposite of common wisdom to discover new perspectives |
| IF Thinking | Expand ideas through "what if" hypothetical scenarios |
| Dialectical Thinking | Synthesize thesis and antithesis into a higher-order conclusion |


## 3. Problem-Solving & Strategic Thinking

| Method | Description |
| --- | --- |
| Hypothesis Thinking | Form a hypothesis first, then refine through verification. Essential for efficient decisions |
| Backward Thinking | Design steps by working backward from the goal |
| Issue Thinking | Correctly define "what should we be thinking about" before solving |
| Framework Thinking | Structure information using frameworks like 3C, SWOT for comprehensive analysis |
| Zero-Base Thinking | Reset all existing assumptions and constraints, think from a blank slate |
| Conceptual Thinking | Capture the essence of things and define them as concepts. Increasingly needed at executive levels |
| Positive-Sum Thinking | Seek solutions that maximize overall benefit rather than win-lose outcomes |


## 4. Analytical & Systems Thinking

| Method | Description |
| --- | --- |
| Systems Thinking | See the whole structure, interdependencies, and feedback loops rather than individual elements |
| Causal Analysis | Identify chains of cause and effect to find root causes |
| Why Thinking | Repeat "why?" five times to reach the true cause (Toyota method) |
| Bottleneck Analysis | Identify and resolve the single biggest constraint limiting overall results |
| Decomposition | Break complex problems into components and address them individually |


## 5. Practical Thinking Tools

| Method | Description |
| --- | --- |
| Visual Thinking | Organize information through diagrams and charts rather than text |
| Debate Thinking | Argue from both sides to clarify the issues at hand |
| PAC Thinking | Verify logic through Premise, Assumption, and Conclusion |
| ABC Theory | Analyze emotions and behavior through Activating event, Belief, Consequence |
| KJ Method | Turn information into cards, group them, and build structure (affinity diagram) |


## Situational Guide

| Situation | Recommended Methods |
| --- | --- |
| Identify root cause of a problem | Why Thinking, Causal Analysis, Bottleneck Analysis |
| Generate new ideas | Lateral Thinking, Analogy Thinking, IF Thinking |
| Organize proposals and plans | Logical Thinking, Framework Thinking, Visual Thinking |
| Challenge assumptions | Critical Thinking, Zero-Base Thinking, Paradoxical Thinking |
| Think from the user's perspective | Design Thinking, Art Thinking |
| Create plans and roadmaps | Backward Thinking, Hypothesis Thinking |
| Understand overall structure | Systems Thinking, Meta Thinking, Conceptual Thinking |
| Improve discussions and decisions | Debate Thinking, Dialectical Thinking, Positive-Sum Thinking |


## 6. Philosophical Thinking Methods


### Ancient Greece

| Method | Key Thinker | Description |
| --- | --- | --- |
| Socratic Method (Maieutics) | Socrates | Draw out knowledge through dialogue and questioning |
| Theory of Forms | Plato | Seek the perfect essence behind observable phenomena |
| Formal Logic (Syllogism) | Aristotle | Foundation of deductive reasoning from premises to conclusions |
| Stoicism | Zeno, Epictetus | Focus on what you can control, let go of what you cannot |
| Epicureanism | Epicurus | Pursue only natural and necessary desires for a tranquil life |
| Skepticism | Pyrrho | Suspend judgment on all assertions |


### Modern Philosophy

| Method | Key Thinker | Description |
| --- | --- | --- |
| Rationalism | Descartes | Build certain knowledge through reason and deduction |
| Empiricism | Locke, Hume | Knowledge originates from sensory experience |
| Dialectics | Hegel | Thesis → Antithesis → Synthesis to advance thought |
| Utilitarianism | Bentham, Mill | Greatest happiness for the greatest number. Judge by outcomes |
| Deontology | Kant | Judge right and wrong by duty and motive, not results |
| Pragmatism | Peirce, Dewey | Truth is judged by "whether it works in practice" |


### Contemporary Philosophy

| Method | Key Thinker | Description |
| --- | --- | --- |
| Phenomenology | Husserl | Set aside assumptions and describe phenomena as they appear in consciousness |
| Existentialism | Sartre, Camus | Humans are free and must define themselves through choices |
| Structuralism | Levi-Strauss | Human thought and society are governed by unconscious structures |
| Post-Structuralism | Derrida, Foucault | Deconstruct binary oppositions and fixed meanings |
| Critical Theory | Habermas | Critically analyze power structures and ideology in society |
| Analytic Philosophy | Wittgenstein | Philosophical problems arise from misuse of language |


### Eastern Philosophy

| Method | Origin | Description |
| --- | --- | --- |
| Zen Thinking | Buddhism (Zen) | Transcend concepts and grasp essence through direct experience |
| The Doctrine of the Mean | Confucianism | Avoid extremes and seek balanced, optimal solutions |
| Yin-Yang Thinking | Chinese Philosophy | See the world through the harmony and cycle of opposing forces |
| Wu Wei (Non-Action) | Taoism | Let go of contrivance and follow the natural flow |


## Relationship Map: Philosophy to Business Thinking

Philosophical thinking methods are the "headwaters" of business thinking methods.\
The table below shows which philosophy influenced each business method.

| Business Thinking | Philosophical Origin | Relationship |
| --- | --- | --- |
| Logical Thinking | Aristotle's Logic | Syllogism and categories are the prototype of structured reasoning |
| Deductive Thinking | Rationalism (Descartes) | Systematized deductive reasoning through reason |
| Inductive Thinking | Empiricism (Locke, Hume) | Methodology for deriving laws from observation |
| Abductive Thinking | Pragmatism (Peirce) | Third form of inference proposed alongside deduction and induction |
| Critical Thinking | Skepticism + Kant | Rooted in the attitude of questioning and critically examining assumptions |
| Dialectical Thinking | Hegel's Dialectics | Thesis-antithesis-synthesis structure applied directly to business |
| Design Thinking | Phenomenology (Husserl) | Observing user experience by setting aside preconceptions |
| Art Thinking | Existentialism (Sartre) | Philosophy of autonomous choice and free creation |
| Systems Thinking | Structuralism + Yin-Yang | Seeing things through overall structure and interrelationships |
| Meta Thinking | Post-Structuralism (Derrida) | Relativizing existing frameworks from a higher level |
| Conceptual Thinking | Plato's Theory of Forms | Extracting the essence and concepts behind phenomena |
| Paradoxical Thinking | Skepticism + Post-Structuralism | Questioning common sense and deconstructing binary oppositions |
| Zero-Base Thinking | Phenomenological Epoche | Bracketing existing judgments and thinking from scratch |
| Framework Thinking | Aristotle's Categories | Prototype of classifying and organizing things into frameworks |
| Hypothesis Thinking | Pragmatism (Dewey) | Inquiry process of hypothesis → experiment → verification |
| Positive-Sum Thinking | Utilitarianism (Bentham, Mill) | Maximizing overall happiness (benefit) |
| Why Thinking | Socratic Method | Dialogue method of repeatedly asking "why?" to reach the essence |
| Debate Thinking | Socratic Method + Dialectics | Clarifying issues by arguing from opposing positions |
| Visual Thinking | Zen (Intuitive Grasp) | Applying intuitive understanding beyond language to visualization |
| KJ Method | Inductive + Structuralism | Inductively structuring individual data points |


## Classification by Higher-Order Concepts

Integrating business and philosophical thinking methods, they converge into three higher-order concepts.

| Concept | Purpose | Business Thinking | Philosophical Origins |
| --- | --- | --- | --- |
| **Logic** — Think Correctly | Organize and verify information | Logical, Critical, Deductive, Inductive, Abductive, Framework, PAC, Debate, Decomposition | Aristotelian Logic, Rationalism, Empiricism, Pragmatism, Skepticism |
| **Creative** — Think Anew | Expand ideas and create value | Lateral, Design, Art, Analogy, Paradoxical, IF, Dialectical, Zero-Base, KJ Method | Phenomenology, Existentialism, Hegelian Dialectics, Post-Structuralism |
| **Panoramic** — Think Holistically | Grasp structure and make decisions | Meta, Abstraction, Systems, Conceptual, Causal, Why, Hypothesis, Backward, Issue, Positive-Sum | Theory of Forms, Structuralism, Utilitarianism, Socratic Method, Taoism, Confucianism |

> Organize with Logic, expand with Creativity, integrate with Panoramic view.\
> Cycling through these three is the fundamental problem-solving cycle in practice.
