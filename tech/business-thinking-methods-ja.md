# ビジネス思考法一覧

更新日: 2026-03-21


## 概要

ビジネス書籍で扱われる主要な思考法を分類・整理した一覧である。\
『思考法図鑑』（翔泳社、60パターン）等の書籍を参考に、目的別に5カテゴリに分類する。


## 1. 基礎思考（考える力の土台）

| 思考法 | 概要 |
| --- | --- |
| ロジカルシンキング（論理的思考） | 筋道を立てて矛盾なく考える。MECE・ロジックツリー等のフレームワークが基盤 |
| クリティカルシンキング（批判的思考） | 前提を疑い、本質を見極める。「So what?」「Why so?」「True?」で検証する |
| 演繹的思考 | 一般的な法則から個別の結論を導く（三段論法） |
| 帰納的思考 | 個別の事実から一般的な法則を導く |
| アブダクション思考 | 結果から最も妥当な仮説を推論する（仮説的推論） |
| メタ思考 | 一つ上の視点から客観的に物事を俯瞰する |
| 抽象化思考 | 具体を抽象化し、別の具体に応用する往復運動 |


## 2. 発想・創造思考

| 思考法 | 概要 |
| --- | --- |
| ラテラルシンキング（水平思考） | 前提にとらわれず、新しい視点やアイデアで自由に発想する |
| デザイン思考 | ユーザーへの共感を起点に、課題を定義し、プロトタイプで検証する |
| アート思考 | 自分の内面の衝動や美意識を起点にゼロから価値を創造する |
| アナロジー思考（類推思考） | 異なる分野の知識を「借りてきて組み合わせる」ことで新しいアイデアを生む |
| 逆説思考 | 常識や定説の逆を考え、新たな着眼点を得る |
| IF思考 | 「もし～だったら」という仮定で発想を広げる |
| 弁証法 | テーゼ（正）とアンチテーゼ（反）を統合し、より高次の結論（合）を導く |


## 3. 問題解決・戦略思考

| 思考法 | 概要 |
| --- | --- |
| 仮説思考 | まず仮説を立て、検証しながら精度を高める。効率的な意思決定に不可欠 |
| 逆算思考 | ゴールから逆算して必要なステップを設計する |
| 論点思考 | 「そもそも何を考えるべきか」という問い自体を正しく設定する |
| フレームワーク思考 | 3C・SWOT 等の枠組みを使って情報を構造化し漏れなく分析する |
| ゼロベース思考 | 既存の前提や制約をすべてリセットし、白紙から考える |
| 概念化思考 | 物事の本質をとらえて概念として定義する。経営層ほど必要とされる |
| プラスサム思考 | 勝ち負けではなく、全体の利益を最大化する解を探す |


## 4. 分析・システム思考

| 思考法 | 概要 |
| --- | --- |
| システム思考 | 個別要素でなく、全体の構造・相互関係・フィードバックループで捉える |
| 因果関係分析 | 原因と結果の連鎖を特定し、根本原因を見つける |
| Why思考 | 「なぜ？」を5回繰り返して真因に到達する（トヨタ方式） |
| ボトルネック分析 | 全体の成果を制約している最大のボトルネックを特定し解消する |
| 要素分解 | 複雑な問題を構成要素に分解し、個別に対処する |


## 5. 実践的思考ツール

| 思考法 | 概要 |
| --- | --- |
| 図解思考 | 文字ではなく図・チャートで情報を視覚化して整理する |
| ディベート思考 | 賛成・反対の両面から議論し、論点を明確にする |
| PAC思考 | 前提（Premise）・仮定（Assumption）・結論（Conclusion）で論理を検証する |
| ABC理論 | 出来事（A）→ 信念（B）→ 結果（C）の構造で感情・行動を分析する |
| KJ法 | 情報をカード化し、グルーピングして構造化する発想法 |


## 場面別の使い分け

| 場面 | 推奨する思考法 |
| --- | --- |
| 問題の原因を特定したい | Why思考、因果関係分析、ボトルネック分析 |
| 新しいアイデアを出したい | ラテラルシンキング、アナロジー思考、IF思考 |
| 企画・提案を整理したい | ロジカルシンキング、フレームワーク思考、図解思考 |
| 前提や常識を疑いたい | クリティカルシンキング、ゼロベース思考、逆説思考 |
| ユーザー視点で考えたい | デザイン思考、アート思考 |
| 計画を立てたい | 逆算思考、仮説思考 |
| 全体構造を把握したい | システム思考、メタ思考、概念化思考 |
| 議論・意思決定を改善したい | ディベート思考、弁証法、プラスサム思考 |

## 6. 哲学的思考法


### 古代ギリシャ

| 思考法 | 代表者 | 概要 |
| --- | --- | --- |
| ソクラテス的問答法（産婆術） | ソクラテス | 対話を通じて相手の中にある知を引き出す |
| イデア論 | プラトン | 現象の背後にある完全な本質を探求する |
| 論理学（三段論法） | アリストテレス | 前提から結論を導く形式論理の基盤 |
| ストア哲学 | ゼノン、エピクテトス | コントロールできることに集中し、できないことを手放す |
| エピクロス哲学 | エピクロス | 自然で必要な欲望のみを満たす穏やかな生活を是とする |
| 懐疑主義 | ピュロン | あらゆる断定を保留し、判断を停止する |


### 近代哲学

| 思考法 | 代表者 | 概要 |
| --- | --- | --- |
| 合理主義 | デカルト | 理性による演繹で確実な知識を構築する |
| 経験主義 | ロック、ヒューム | 知識は感覚的経験から生まれる |
| 弁証法 | ヘーゲル | 正→反→合で思考を発展させる |
| 功利主義 | ベンサム、ミル | 最大多数の最大幸福。結果の有用性で判断する |
| 義務論 | カント | 結果ではなく動機と義務で善悪を判断する |
| プラグマティズム | パース、デューイ | 真理は「実践で有用かどうか」で判断する |


### 現代哲学

| 思考法 | 代表者 | 概要 |
| --- | --- | --- |
| 現象学 | フッサール | 先入観を排し、意識に現れる現象そのものを記述する |
| 実存主義 | サルトル、カミュ | 人間は自由であり、自ら選択して生きる |
| 構造主義 | レヴィ＝ストロース | 人間の思考や社会は無意識の構造に規定されている |
| ポスト構造主義 | デリダ、フーコー | 二項対立や固定的な意味を解体する |
| 批判理論 | ハーバーマス | 社会の権力構造やイデオロギーを批判的に分析する |
| 言語分析哲学 | ウィトゲンシュタイン | 哲学の問題は言語の誤用から生じると考える |


### 東洋哲学

| 思考法 | 起源 | 概要 |
| --- | --- | --- |
| 禅的思考 | 仏教（禅宗） | 概念を超え、直接的な体験と気づきで本質を悟る |
| 中庸 | 儒教 | 極端を避け、バランスの取れた最適解を探る |
| 陰陽思考 | 中国哲学 | 対立する二つの力の調和と循環で世界を捉える |
| 無為自然 | 道教 | 作為を捨て、自然の流れに従う |


## 哲学的思考とビジネス思考の関係マップ

哲学的思考法はビジネス思考法の「源流」にあたる。\
以下の表は、各ビジネス思考法がどの哲学から影響を受けているかを示す。

| ビジネス思考法 | 源流となる哲学 | 関係性 |
| --- | --- | --- |
| ロジカルシンキング | アリストテレスの論理学 | 三段論法・カテゴリー論が論理的思考の原型 |
| 演繹的思考 | 合理主義（デカルト） | 理性による演繹推論を体系化 |
| 帰納的思考 | 経験主義（ロック、ヒューム） | 経験・観察から法則を導く方法論 |
| アブダクション思考 | プラグマティズム（パース） | パースが演繹・帰納に加えて提唱した第三の推論 |
| クリティカルシンキング | 懐疑主義 + カント | 前提を疑い批判的に検証する姿勢の源流 |
| 弁証法 | ヘーゲルの弁証法 | 正→反→合の構造がそのままビジネスに応用 |
| デザイン思考 | 現象学（フッサール） | 先入観を排してユーザー体験を観察する手法 |
| アート思考 | 実存主義（サルトル） | 主体的な選択と自由な創造の哲学 |
| システム思考 | 構造主義 + 陰陽思考 | 全体の構造・相互関係で捉える見方 |
| メタ思考 | ポスト構造主義（デリダ） | 既存の枠組みを一段上から相対化する |
| 概念化思考 | プラトンのイデア論 | 現象の背後にある本質・概念を抽出する |
| 逆説思考 | 懐疑主義 + ポスト構造主義 | 常識を疑い、二項対立を解体する |
| ゼロベース思考 | 現象学の判断停止（エポケー） | 既存の判断を括弧に入れ、白紙から考え直す |
| フレームワーク思考 | アリストテレスのカテゴリー論 | 物事を分類・整理する枠組みの原型 |
| 仮説思考 | プラグマティズム（デューイ） | 仮説→実験→検証の探究プロセス |
| プラスサム思考 | 功利主義（ベンサム、ミル） | 全体の幸福（利益）を最大化する思想 |
| Why思考 | ソクラテス的問答法 | 「なぜ？」を繰り返して本質に迫る対話法 |
| ディベート思考 | ソクラテス的問答法 + 弁証法 | 対立する立場から論点を明確にする |
| 図解思考 | 禅的思考（直観的把握） | 言語を超えた直観的理解を視覚化に応用 |
| KJ法 | 帰納的思考 + 構造主義 | 個別データを帰納的に構造化する手法 |


## 上位概念による分類

ビジネス思考法と哲学的思考法を統合すると、3つの上位概念に収束する。

| 上位概念 | 目的 | ビジネス思考法 | 源流となる哲学 |
| --- | --- | --- | --- |
| **論理系** — 正しく考える | 情報を整理・検証する | ロジカルシンキング、クリティカルシンキング、演繹的思考、帰納的思考、アブダクション、フレームワーク思考、PAC思考、ディベート思考、要素分解 | アリストテレス論理学、合理主義、経験主義、プラグマティズム、懐疑主義 |
| **創造系** — 新しく考える | 発想を広げ価値を生む | ラテラルシンキング、デザイン思考、アート思考、アナロジー思考、逆説思考、IF思考、弁証法、ゼロベース思考、KJ法 | 現象学、実存主義、ヘーゲル弁証法、ポスト構造主義 |
| **俯瞰系** — 全体を見て考える | 構造を把握し意思決定する | メタ思考、抽象化思考、システム思考、概念化思考、因果関係分析、Why思考、仮説思考、逆算思考、論点思考、プラスサム思考 | イデア論、構造主義、功利主義、ソクラテス問答法、道教（無為自然）、儒教（中庸） |

> 論理系で整理し、創造系で広げ、俯瞰系で全体を統合する。\
> この3つの思考を循環させることが、実務での問題解決の基本サイクルである。
