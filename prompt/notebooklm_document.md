NotebookLMで作る論文ビジュアル構造化サマリ
第2弾はスマホで見やすい"縦長"レイアウト

上から「CQ→方法→結果→結論」でスラスラ頭に入ります。そして最大のメリットは患者フローを横一列に配置できるところ。

使い方はレイアウトを「縦向き」に設定して以下のプロンプトをコピペするだけ👉

---以下をコピペして使う---

全体デザイン設定:
  テーマ: "Clinical Paper Infographic"
  用途: "臨床論文・レビュー論文の要点を、縦長1枚（印刷/スライド）で提示する"

  レイアウト:
    形式: "A4縦 または 9:16 縦長"
    構成: "最上部ヘッダー + 下部3パネル縦積み（上: Setting / 中: Key Results / 下: Conclusion）"
    高さ配分の目安:
      ヘッダー: "約12–15%"
      上段パネル_SETTING: "約25–28%"
      中段パネル_KEY_RESULTS: "約45–53%"
      下段パネル_CONCLUSION: "約10–15%"
    パネル間の区切りデザイン:
      方式: "各パネルの下端中央に小さなV字ノッチ（凹み）を入れてセクション連続性を表現"
      ノッチ幅: "パネル幅の6–10%"
      ノッチ高さ: "12–18px 相当"
      ノッチ色: "次のパネル背景色に合わせる（境界を自然に見せる）"

  ビジュアル・アイデンティティ:
    背景色全体: "#E5E7EB 〜 #F3F4F6 のグレー調"
    ヘッダー背景色: "#374151"

    文字色:
      メイン: "#111827"
      サブ: "#4B5563"
      ヘッダー文字: "#FFFFFF"

    アクセントカラー: "#2563EB"

    パネル配色:
      上段パネル背景: "#F9FAFB"
      中段パネル背景: "#FFFFFF"
      下段パネル背景: "#F3F4F6"

    図形スタイル（SETTING用）:
      ボックス:
        角丸: "8–10px"
        枠線: "1px #CBD5E1"
        塗り: "白〜薄グレー（配色は固定しない。可読性優先）"
        内側余白: "10–12px"
      矢印:
        形状: "直線矢印 →（必要に応じて分岐矢印）"
        色: "#9CA3AF"
        太さ: "2px相当"
      タイムライン帯:
        形状: "横長の矢印帯（→）または帯ラベル"
        位置: "フロー図の最上段"
        目的: "期間/時間の流れを左→右に示す（例：Index/Exposure definition → Follow-up）"

    アイコン:
      スタイル: "カラーのシンプルなフラットアイコン"
      配置方針: "SETTINGでのみ最小限で使用する（KEY_RESULTS/CONCLUSIONでは原則使用しない）"
      サイズ目安: "22–28px"

  タイポグラフィ:
    タイトル: "太めのサンセリフ体（日本語・英語両対応）"
    セクション見出し: "中サイズ・太字・オールキャップ（SETTING, KEY RESULTS, CONCLUSION）"
    本文: "読みやすいサンセリフ体、行間はやや広め"
    強調表現: "太字またはアクセントカラーで示す"

ページ構成:
  - セクション: "Header（Clinical Question + 書誌情報）"
    位置: "ページ最上部・全幅（高さ 12–15%）"
    内容:
      - "日本語で一言（結論に直結する問いの形にする）"
      - "Bibliographic info（英語）：First author, Full paper title, Journal, Year（英語のまま）"
    デザイン:
      - "背景色 #374151、文字は白。"
      - "上段にClinical Question（日本語）を大きめ、下段に書誌情報（英語）を小さめで配置。"
      - "ヘッダー下端に細い明るいグレーの水平線を入れて本文と区切る。"

  - セクション: "上段パネル – SETTING（研究設計/コホートフロー図：文章仕様で再現）"
    位置: "ヘッダー直下（高さ 25–28%）"
    背景: "#F9FAFB"
    見出し:
      表記: "SETTING"
      スタイル: "濃いグレーの見出しバー + 白文字"

    レイアウト:
      表示形式: "研究フロー（箱＋矢印）"
      全体構造: "左→右に流れる3エリア構成（Data & Eligibility → Cohort definition → Follow-up & Outcome）"

      分岐要件:
        適用条件:
          - "コホート数が2以上の場合は、中央エリア（Cohort definition）から右方向にラインを分岐させる"
        分岐位置:
          - "分岐は中央エリアの右端（Cohort definitionボックスの直後）で行う"
        分岐表現:
          - "各コホートは独立した矢印ライン（レーン）として右エリアへ接続する"
          - "各レーンの開始点にコホート名ラベル（例：Group A / Group B / Group C）を付与する"
        症例数:
          - "Cohort definition直後の矢印（分岐なら各レーン）に N=… を付ける（不明なら省略）"
        右エリア側の対応:
          - "Follow-up & Outcome のボックスは原則共有（共通）として配置し、各レーンは同じボックス列に合流してもよい"

    内容ブロック（テンプレ）:
      左エリア_Data_Eligibility:
        タイトル: "DATA & ELIGIBILITY"
        表示: "大きめの単一ボックス（箇条書き）"
        推奨項目:
          - "Data source（例：SEER-Medicare / claims / registry）"
          - "期間（data period）"
          - "対象疾患/条件（disease/condition）"
          - "主要な選択基準（inclusion）"
          - "主要除外（exclusion）"

      中央エリア_Cohort_definition:
        タイトル: "COHORT DEFINITION"
        表示: "中央ボックス＋分岐矢印（2〜3コホート）"
        表記例:
          - "Index date / time zero（起点）"
          - "Exposure / group definition（群の定義）"
          - "Comparator/group B definition（比較群の定義）"
        調整手法（該当する場合のみ）:
          表示条件: "論文で交絡調整手法として明記されている場合に限る"
          追記候補:
            - "PS matching (1:x)"
            - "IPTW / weighting"
            - "adjustment"
          表示位置: "中央ボックスの下に小さなサブラベル、または右エリアへ入る直前の細字1行"

      右エリア_Outcome:
        タイトル: "OUTCOME"
        内容例:
          - "Primary outcome"
          - "ANALYSIS"

    ノッチ:
      有無: true
      位置: "下端中央"
      形状: "V字凹み"
      色: "#FFFFFF"

  - セクション: "中段パネル – KEY RESULTS"
    位置: "上段パネルの下（高さ 45–53%）"
    背景: "#FFFFFF"
    見出し:
      表記: "KEY RESULTS"
      スタイル: "濃いグレーの見出しバー + 白文字"
    内容ブロック:
      説明:
        - "重要な結果を図表中心で表示する。"
        - "可能な限り“図種を変えない”（例：Kaplan–MeierはKaplan–Meierのまま、棒グラフ等に変換しない）。"
        - "必要に応じて複数のグラフを配置するが、視覚的要素を優先する。"
    デザイン:
      - "右カラムの背景は #FFFFFF。"
      - "グラフは横長レイアウトを基本とし、テキストは最小限に抑える。"
    ノッチ:
      有無: true
      位置: "下端中央"
      形状: "V字凹み"
      色: "#F3F4F6"

  - セクション: "下段パネル – CONCLUSION"
    位置: "最下部（高さ 10–15%）"
    背景: "#F3F4F6"
    見出し:
      表記: "CONCLUSION"
      スタイル: "濃いグレーの見出しバー + 白文字"
    内容ブロック:
      記載:
        - "結論を日本語で1〜2文"
        - "必要なら注意点/臨床的含意を1-2行だけ追加"
      デザイン:
        - "重要語を太字またはアクセントカラーで強調。"

デザインルール:
  - "縦長（A4縦 or 9:16）。"
  - "ヘッダーは『Clinical Question（日本語・一言）』＋『書誌情報（英語：筆頭著者、正式タイトル、雑誌名、年）』の2段構成。"
  - "セクション見出しは英語で統一（SETTING / KEY RESULTS / CONCLUSION）。本文は日本語で要約する。"
  - "視覚的主役はKEY RESULTSセクションとする。"
  - "色数は抑え、配色で情報を詰め込みすぎない（コントラストと可読性優先）。"
  - "印刷/スマホ閲覧でも読める文字サイズを確保する。"
